# Marketing Site - Bemi Creatives

Static HTML landing page for the Online Menu System.

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment

This is a static site. Simply upload the contents to any web host or CDN.

### Vercel
- Set root directory to `marketing-site`
- No build command needed
- Serves `index.html` directly

## Structure

```
marketing-site/
├── index.html          → Main landing page
├── assets/
│   └── Images/         → All marketing images
└── package.json
```

## Features

- Responsive design
- Mobile menu
- Smooth scrolling navigation
- Contact form integration (Formspree)
- Image slider
- Client showcase

## Note

This is a simplified static version converted from the original `Marketing.jsx` React component. If you need the full interactive experience with all sections, you can expand the HTML file or use the original React component in a Next.js app.
